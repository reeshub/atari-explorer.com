Personal Computers :: 8-Bit Emulation :: Quick Start Guide

This Quick Start guide is aimed at the very beginner, and is designed to get your Atari 800 Windows Plus Emulator up and running quickly.  We recommend you read the included HELP file further for more in depth information, and to get the most from the program.

Lets go!  Assuming you have already downloaded the two most important files (the emulator and the 8-Bit ROMs), you should create a new directory on your hard drive using Windows Explorer.  This can be called anything you want, but a good idea is to keep folder names appropriate and simple.  Once this is done, "unzip" the 2 downloaded files to the new folder.  If you haven't got a copy of WinZip, go here...

! Important - Atari800Win+ requires Direct-X 7.0 or higher to run at its best, please ensure your OS has this driver installed.  If in doubt, get the latest version from the Microsoft website.

Once the files have been extracted (un-zipped) to your new folder, you will see an icon similar to the one above, it looks like an Atari 800XL computer.  This is the .EXE file for Atari800Win+.  Double click on this icon to start the program.

During the very first installation, you will be asked a number of questions. The first question asks you if you want shortcuts to the emulator, the second question is to confirm the location of your ROM files, and the last asks what file extensions you want to associate with the emulator.  The most important question is the location of your Atari -Bit ROM files.  Just point the program to the appropriate folder where each of the ROM files are contained (which should be in the new folder you created and extracted the files to).  The ROM files are required for the emulator to work properly.

When the program asks you to "associate" files for the emulator, select all.  You should be presented with a "Finish" button, press this, and we can proceed.

You should be presented with a screen similar to the one above if installation was successful.  Don't worry if you think you screwed the installation part up, because you can always do it again, even after the program has loaded, by going to the HELP option, and selecting the setup wizard.

Let's get accustomed to the program interface, which has been made nice and easy to use.  You will learn more about the program the more you use it, and the HELP file included is very useful to get the most from Atari800WinPlus, especially when trying out more complex operations.  But for now, we'll get to grips with the basic operation, and the most common options we will use are FILE and ATARI.  When you get more familiar with the program, Atari800WinPlus has "state indicators" which will allow you to change and set other features, but for now, ignore them.

So, we are ready to load our first game!  Pole Position was included in the Emulator files you downloaded, so lets try that one!  The first thing we will do is ensure the machine type is set-up correctly for Pole Position.  When you load future programs, you will need to change certain settings if the program fails to load.  This can be a process of trial and error, but the most common errors concern the machine type, memory size, or if you have selected options such as fast disk access or different system speeds.

For Pole Position, we will set the machine type to OS-B, which is the Atari ROM, Operating System Rev. B, one of the most common.  We should also check the memory is set to 48K (Memory Size) and the Video System is set to PAL (50Mhz), this would only change if the game was a U.S. NTSC version at 60Mhz.

! If you cannot see any machine options, you need to confirm the ROM locations again, go to HELP, and select the set-up wizard.

As most of the program options do not need changing at this point, we can continue and load the actual game file.

Go to FILE, and select the ATTACH DISK option.  As you can see above, you are presented with 8 disk drives, but we only need one for Pole Position!  Click on DRIVE 1.

A windows file box will open, and you should go to the folder you created earlier, this will contain your file for Pole Position.  Open this file.

The game Pole Position is now loaded into Disk Drive 1.  Before we run the game, we'll just check our INPUT options.  You can use a real joystick with the emulator, but to get playing quickly, we'll use the PC keyboard for now.  Go to INPUT, and select JOYSTICKS.

To use the PC keyboard, select the option above in Row 1.  We are now ready to run Pole Position.  The keys associated with the game will be Right Control (FIRE BUTTON) and the Keypad as the directional input.  Click OK and proceed.

! On the Atari 8-Bit computers there was 4 important buttons.  START, SELECT, OPTION and RESET.  The same buttons apply to the emulator, F2 = Option F3 = Select F4 = Start F5 = Reset (if a game doesn't reset, or a new program fail to load, press SHIFT+F5 for a cold reset).

So, lets press RESET on the PC keyboard (F5) and see what happens...

The program has loaded!  Some programs take a few seconds to load, and some take a minute or two.  When a program is loading you will see a small green number flashing in the bottom right of the emulator screen, this shows the loading progress just like a real disk drive indicator light.  Press START to play (or SELECT and OPTION to change game settings).

Atari800WinPlus is a fantastic emulator, and with more practice you can play practically every 8-Bit game released.  The program has many exciting options, such as full screen, sound and video recorders, network play options and more...  have fun, and always read the manual ;-)
