Component:  Description:
 

 System Processors:

 

T-800 (20Mhz)and 68000 (8Mhz) connected via a Transputer link
 Graphics Processor:

"Blossom" Video CPU
 

32 x 32 Pixel "mask"
 

16 bit wide functions for source and destination
 

4 linear comparisons for each pixel per cycle
 

A Square area can fill 120 megapixels p/s
 

A Line can be drawn at up to 32 megapixels p/s
 

A Trapezoid can be drawn at up to 32 megapixels p/s
 

X-Windows compatible
 System Memory:

T-800/20 host processor has 4MB's of dynamic RAM (120ns) expandable to 16MB's.  The 68000 has 512Kb, expandable to 1MB
 Video Memory:

1Mb of fast dual-port dynamic RAM (100ns) with ports connected to the main bus and the video bus
 I/O:

Controlled by the 68000, which runs independently of the T-800 host processor
 Monitors:

NEC Multisync Plus/XL, Philips M1964 and Hitachi 4616 (D-BB-3)

 Optional Peripherals:

Atari CDAR-504 CD-ROM Drive
 

Atari Megafile Hard Disk Drives
 Expansion Slots:

The T-800 has Four (4) expansion slots and the 68000 has One (1)
 Processor Expansion:

An additional 16 T-800 expansion cards can be added internally with Transputer Farm Cards.  An unlimited number can be added externally.
