Component:  Description:
 

 Main Processors:

  

"Mikey" and "Suzy" - 16-Bit Custom CMOS running at 16Mhz

 "Mikey":

MOS 65C02 processor running at up to 4MHz
 

8-bit CPU, 16-bit address space
 Sound System:

4 channel sound, 8-bit DAC for each channel
 

Stereo with panning (mono for original Lynx)
 Video for LCD:

Video DMA driver for LCD display
 

4096 colour (12-bit) palette
 

16 simultaneous colours (4 bits) from palette per scanline
 Misc. Functions:

System timers
 

Interrupt controller
 

UART (for ComLynx) - 300.5K to 62.5K baud
 

512 bytes of bootstrap ROM and game-card loading ROM
 

 
 "Suzy":

Graphics engine and math co-processor
 Graphics Engine:

Blitter unit (bit-map block transfer)

 

Hardware drawing support
 

Unlimited number of high-speed sprites with collision detection
 

Hardware high-speed sprite scaling, distortion, and tilting effects
 

Hardware decoding of compressed sprite data
 

Hardware clipping and multi-directional scrolling
 

Variable frame rate (up to 75 frames/second)
 

160 x 102 standard resolution (16,320 addressable pixels)
 

Artificially high resolution of 480 x 102

 Math co-processor:

Hardware 16-bit multiply and divide (32-bit answer)
 

Parallel processing of single multiply or divide instruction
 

 
 Onboard Memory:

64K (half a megabit) of 120ns DRAM
 LCD:

3.25" x 1.88" approx
 Buttons:

1 X directional (joypad) controller (8-Way)
 

2 X 2 (4) fire buttons (A and B)
 

2 X option buttons (OPTION 1 and OPTION 2)
 

Pause button
 

Power on button
 

Power off button
 

Backlight on/off button (Lynx 2 only)
 Controls:

Volume wheel
 

Brightness wheel
 Other:

Power on LED (Red) - (Lynx 2 only)
 Ports:

Headphones - mini-DIN 3.5mm stereo (Lynx 2) mono on the Lynx 1
 

ComLynx (multiple unit communications)
 

Power (9V DC, 1 A)
 

Game card slot
 

Battery compartment (6 X AA size)
 Speaker:

2 inch internal
 Dimensions:

10.75" x 4.25" x 1.5"  - Lynx 1
 

9.25" x 4.25" x 2" - Lynx 2
