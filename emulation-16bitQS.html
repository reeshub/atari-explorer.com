Personal Computers :: 16-Bit Emulation :: Quick Start Guide

This Quick Start guide is aimed at the very beginner, and is designed to get your STeem Emulator up and running quickly.  We recommend you read the included HELP files further for more in depth information, and to get the most from the program.

Lets go!  Assuming you have already downloaded the two most important files (the emulator and the 16-Bit ROMs), you should create a new directory on your hard drive using Windows Explorer.  This can be called anything you want, but a good idea is to keep folder names appropriate and simple.  Once this is done, "unzip" the 2 downloaded files to the new folder.  If you haven't got a copy of WinZip, go here...

! Important - STeem requires Direct-X 7.0 or higher to run at its best, please ensure your OS has this driver installed.  If in doubt, get the latest version from the Microsoft website.

Once the files have been extracted (un-zipped) to your new folder, you will see an icon similar to the one above, its an Atari Fuji logo.  This is the .EXE file for STeem 2.4.  Double click on this icon to start the program.

When you install STeem for the very first time, it will ask you a number of questions, including the location of your 16-Bit ST ROMS (which we extracted earlier, just point the program to the respective folder on your computer).  These questions are all straight forward, and don't worry if you make a mistake here, as you can always change the settings once the program has loaded.

If the program has loaded correctly, you should be presented with a screen similar to the one above, the famous green ST Gem desktop.

Lets take a quick tour around the STeem window.  On the top, you will see a row of icons, these represent the main function areas of the emulator.  The most commonly used icons are "Run", "Reset", "ST Configuration" and "Disk Manager".  Note, that the first Five icons have double settings, and Right-Mouse clicking will provide a different option.  If in doubt, just hover over each button to discover the option types.

Before we load an actual ST program, you will notice that we are going to configure the ST type within the emulator.  Sometimes you will try a program and it will "crash" the ST, or in other words, the program will fail to load properly.  This can be due to a number of reasons, such as the memory type, TOS ROM revision or perhaps, a bad program image.  Again, just because a program doesn't load first time around, it pays to play around with the settings here.

! STeem will not run unless it can see the ST ROMS.  These are the ST Operating System ROMS that contain the essential data to make the ST operate properly.  Unless you have pointed STeem to the ROMs on your computer, your emulator cannot function.

We are going to set up our STeem emulator to play a game of Buggy Boy, which is included in the files you extracted earlier.  Click on the "ST Configuration" icon (the third icon from the right), a menu will appear like the one above.  The settings we are going to choose are TOS 1.62 with 1MB RAM (although other setting may also work for this particular program).

When we have set up our ST Configuration, we will ensure we can play the game using the PC keyboard.  STeem will allow you to set-up many different input options, including a Joystick, but for this introduction, we'll use the keyboard.  Click on the "Joystick Configuration" icon (the second icon from the right), a menu will appear like the one above.  The original ST computer had 2 joystick ports, port 0 was for the mouse, and port 1 was for the joystick.  Alternatively, 2 joysticks could be used for 2 player games.

We have set up the Joystick Configuration to use the mouse when the SCROLL LOCK button is ON, and to use a joystick when the NUM LOCK is OFF.  This way we can use the GEM desktop with the mouse when we need to, and switch to the Joystick mode (using the keyboard in this case) when we want to play a game.

OK, now let us load the game Buggy Boy into the emulator.  Original ST Disks have been copied into "images" for use on the emulator, and we use the "Disk Manager" to load them.  Click on the "Disk Manager"  icon (the first icon from the right), a menu will appear like the one above.

Depending on where you told STeem the location of your ST Images, you should use the Disk Manager to find the game "Buggy.st".  If the default location doesn't show this file, locate it on your hard drive using the folder direction buttons - the game will be in the folder you created earlier.

To load any game or other ST Image, you can either drag the respective icon to the "A" drive (Floppy Disk), or double click on it to run.  If you drag the disk to the "A" drive, you will need to go back to the main STeem window and press the Green "Run" icon (first icon on the left).  When an ST Image begins to load in STeem, you will see STeem display "on screen" information, just like the picture above...

! STeem can unzip ST disk images on its own, so there is no need to unzip ST disk images.

You will also see the disk drive activity light activate when a program is loading.  Some games take a few seconds to load, while others can take longer.  STeem has the ability to speed up this process by clicking on the "Fast Forward" icon (second icon on the left), but not all programs may like this, and could crash.  Once the game has loaded, your ready to play! 

As we have set up the PC Keyboard to emulate an ST Joystick, use the Right Control Key for "Fire", and the Keypad Arrows to move respectively.  In this game, the Space Key will change your gears from Hi to Low, and the Up and Down arrows will change your speed.

You can also resize your STeem window by dragging a corner of the window to any size (this may slow the program down, depending on you PC configuration).  STeem will also run in Full Screen mode, which you can set-up in the "Options" menu, but before making any adjustments there, take note of your current set-up, so you can always reset it back if things don't work out properly.  Experiment with your set-up only when you are more familiar with STeem, and use the "General Info" menu to learn more about these options.

So, you've been playing a genuine ST game, and isn't it great!  Buggy Boy is a definite classic.  The ST has thousands of programs, and STeem can emulate most of them perfectly, as you can see below, even ST applications will work fine under STeem...

STeem has many more features for you to explore, its a fantastic program which will give you years of pleasure, have fun, and always read the manual ;-)
